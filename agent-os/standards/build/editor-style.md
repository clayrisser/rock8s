# Editor & Style Config

Defined in `.editorconfig` at repo root.

## Indentation

| File type | Style | Width |
|-----------|-------|-------|
| Default (`*`) | spaces | 2 |
| Shell (`*.sh`) | spaces | 4 |
| Makefile, Mkpmfile, `*.mk` | tabs | 4 |
| Markdown (`*.md`) | spaces | 2, no trailing whitespace trim |

## Line endings

LF everywhere. Final newline inserted. Trailing whitespace trimmed (except markdown).

## Charset

UTF-8 for all files.
